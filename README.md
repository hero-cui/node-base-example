# node-base-example

极简的 node 项目模板，包含了 node + express + jsonwebtoken + mysql2 的基本配置。

已经处理了跨域问题，已经处理了 token 验证问题，已经处理了数据库连接，以及文件上传等问题。

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run start:dev
```

### Compiles and minifies for production

```
npm run start:prod
```
